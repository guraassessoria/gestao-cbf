# GitHub + Vercel

## GitHub
```powershell
git init
git add .
git commit -m "feat: balancete mvp api"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/balancete-mvp-api.git
git push -u origin main
```

## Vercel
1. Importar o repositório no dashboard da Vercel
2. Framework preset: Other
3. Definir as variáveis de ambiente
4. Deploy

## Atualizações
Qualquer push novo na branch principal gera deploy automático na Vercel.
